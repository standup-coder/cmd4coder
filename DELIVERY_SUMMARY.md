## 🎓 使用示例

> **注意**: 以下命令可使用 `go run ./cmd/cli` 替代 `cmd4coder` 执行。

### 快速上手
```bash
# 1. 查看所有分类
go run ./cmd/cli categories -d ./data

# 2. 列出某分类下的命令
go run ./cmd/cli list "Database" -d ./data

# 3. 查看具体命令
go run ./cmd/cli show "docker run" -d ./data

# 4. 搜索相关命令
go run ./cmd/cli search kubernetes -d ./data
```
